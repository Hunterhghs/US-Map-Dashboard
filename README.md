# US Map Dashboard — Economy & Development 2000–2025

**Interactive multidimensional economic dashboard covering all 50 US states + DC.**

Built by [H Heuristics](https://hheuristics.com) — market research & economic development consulting.

---

## Overview

Explore 26 years of economic data (2000–2025) across 9 key metrics:

| Metric | Source |
|--------|--------|
| **GDP** ($ billions) | Bureau of Economic Analysis |
| **GDP Growth** (% annual) | Bureau of Economic Analysis |
| **GDP per Capita** ($) | BEA / Census Bureau |
| **Population** (millions) | Census Bureau |
| **Unemployment Rate** (%) | Bureau of Labor Statistics |
| **Median Household Income** ($) | Census Bureau |
| **Poverty Rate** (%) | Census Bureau |
| **Housing Price Index** (2000=100) | FHFA |
| **Educational Attainment** (% Bachelor's+) | Census Bureau ACS |

---

## Features

- 🗺️ **Interactive choropleth map** — Leaflet.js with 9 switchable metric overlays
- ⏱️ **Animated timeline** — Play/pause through 2000–2025 with speed controls
- 📊 **KPI dashboard** — National aggregates with year-over-year changes
- 📈 **Bar chart** — Top 12 states ranked by selected metric
- 🔍 **Searchable state list** — Sort by rank, search by name, fly-to on click
- 🏷️ **Region filtering** — Northeast, Midwest, South, West
- 🎨 **Dark professional theme** — H Heuristics design system
- ⌨️ **Keyboard shortcuts** — Arrow keys, Space, B, F, Home, End
- 📱 **Fully responsive** — Desktop, tablet, and mobile layouts
- 🔲 **Fullscreen mode** — Immersive map experience

---

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `←` `→` | Previous / Next year |
| `Space` | Play / Pause animation |
| `B` | Toggle sidebar |
| `F` | Toggle fullscreen |
| `Home` | Jump to 2000 |
| `End` | Jump to 2025 |

---

## Tech Stack

- **Leaflet.js 1.9.4** — Interactive maps with Canvas renderer
- **Chart.js 4.4.7** — Sidebar bar chart
- **CARTO dark basemap** — Professional dark tiles
- **Vanilla HTML/CSS/JS** — Zero build step, fully self-contained
- **GitHub Pages** — Instant deployment

---

## Data Notes

- Data synthesizes official statistics (BEA, Census Bureau, BLS, FHFA) with modeled estimates
- 2024–2025 values are projections based on current trends
- GDP figures are nominal (current USD)
- Housing Price Index: 2000 = 100 base
